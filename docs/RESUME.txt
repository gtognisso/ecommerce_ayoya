                        DÉPLOIEMENT RAPIDE
================================================================================

OPTION 1: Docker Compose (Recommandé)
────────────────────────────────────────────────────────────────────────────
cd /home/claude-dev/qrcode/ayoya-ecommerce/docs
./start.sh

Puis accès à:
  http://localhost:8080           (Index)
  http://localhost:8080/admin/    (Guide Admin)
  http://localhost:8080/logistics/ (Guide Logistique)

OPTION 2: Docker manuel
────────────────────────────────────────────────────────────────────────────
docker build -t ayoya-docs .
docker run -p 8080:80 ayoya-docs

OPTION 3: Nginx direct
────────────────────────────────────────────────────────────────────────────
sudo cp -r admin logistics /usr/share/nginx/html/
sudo cp nginx.conf /etc/nginx/nginx.conf
sudo systemctl restart nginx

OPTION 4: Python (développement)
────────────────────────────────────────────────────────────────────────────
python3 -m http.server 8080
